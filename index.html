What it does:

It has some built-in attributes which can change the scrollbar of your website and give it a beautiful look
You can use it by adding some attributes in the <body> tag


Attributes:

  1. psw
    psw attribute is used to change the width of the scrollbar.
    Example: <body psw="15"> makes the width of the scrollbar 15 pixels

  2. pscol
    pscol attribute is used to change the color of the scrollbar.
    Example: <body pscol="blue"> makes the color of the scrollbar blue

  3. psbg
    pscol attribute is used to change the background-color of the scrollbar.
    Example: <body psbg="red"> makes the background-color of the scrollbar red

  4. psround
    psround attribute is used to make the borders of the scrollbar round
    Example: <body psround="true"> makes all the borders of the scrollbar round

  5. pstemp
    pstemp attribute is used when you want to start from a built-in template, you can also add your own styles to a built-in template.
    Templates:

    • youtube
    • default
    • dark
